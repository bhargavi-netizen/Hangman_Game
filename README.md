# Hangman Game (CodeAlpha Internship Task)

## Description
A simple text-based Hangman game written in C. The player guesses a word one letter at a time.

## Features
- Random word selection
- 5 predefined words
- 6 incorrect attempts allowed
- Console-based interface

## Concepts Used
- Arrays
- Strings
- Loops
- Conditional statements

## How to Run
1. Compile: gcc hangman.c -o hangman
2. Run: ./hangman
